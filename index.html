👩🏽‍🚀 Projects

Cyber Threat Intelligence Tool - GE Aviaton

As apart of my senior year capstone, I developed a cyber threat intelligence CLI tool in python called Cyber Turtle. This tool is comprised of three main components, a JSON wordlist of relevant terms to the user, a content analyzer to read in reports, and a similarity calculator. The tool uses the TFIF algorithm to calculate a normalized score for each document that is read in based on a constantly changed base line and then report it to the user to determine if it is worth reading for TPI.
